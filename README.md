# Phishing Link Detector

## Setup

1. Install dependencies:
    ```
    pip install -r requirements.txt
    ```

2. Train the model:
    ```
    cd src
    python train_model.py
    ```

3. Run the web app:
    ```
    python app.py
    ```

4. Open your browser at [http://127.0.0.1:5000](http://127.0.0.1:5000)
